# How to draw a good street map

A recipe brought to you by Carlos Azeredo Mesquita
and Maria João Macedo, the masterminds behind
USE-IT Porto, Funchal and Guimarães.

1. This will be a tutorial on how to draw a street map
    that is good and easy to read, and easy to update.
    It will require quite some work, but once you’ve
    got it, you will use it for many years to come.
2. You will need: internet in order to access Google
    Maps, and Adobe Illustrator.
3. Plenty of patience!

## ! DISCLAIMER!

This tutorial is meant for designers and people
experienced with the aforementioned software
and vector drawing. Creating a good street map is
not really a DIY easy thing as it might come across
in this document, so better leave it for those who
know how to do it.


# 1. Drawing the streets

Rationalize

- it’s a walking map: large streets for cars are not
    the most important
- it’s non-topographical: you can simplify the
    shapes and curves, you can make narrow streets
    wider (mostly for the old town areas)

Hierarchize

- make the differences between the streets obvious

but use no more than 4 stroke widths

Simplify

- make curves smoother, corners more geometric,
    roundabouts actually round
- no need to represent buildings inside the blocks

Include some subtleties

- dashed lines for tunnels, double outlines for
    walks, dotted outlines, marking the inside of
    squares etc., help make the map more legible

# 2. Typing the street names

Don’t be lazy!

- name as many streets as you can

Make it easy to read

- use ALL CAPS only
- type size must fit the stroke width and must be in
    the middle of it
- don’t make type too small (min 4pt, max 12pt)

Choose wisely

- one typeface is enough
- avoid serif or decorative fonts
- use a condensed font to get more letters on a
    small space
- you can increase spacing between letters to get
    more coverage
- use 100% tint (it prints better)

# 3. Colouring the map

```
Streets
```
- keep the streets white or light in colour (so that
    people can draw on them)
- use an outline with a darker colour, very thin
    stroke – about 0,25pt or 0,3pt – but 100% tint, in
    good balance with the stroke of typeface of the
    street names.
Backgrounds
- give the background a neutral colour, no need for
    strong contrast
- represent squares as open spaces
- parks should be distinguished with a different
    colour filling
- use textures as little as possible, at most for
    water and green areas only

# 4. Overall balance

```
From top to bottom
```
- Apprehend the city shape in one glance (the most
    important street must pop-out right away)
- The dots and buildings come next in importance
- Additional information (transport lines, tunnels,
    metro stops, city neighbourhoods, walking tours,
    etc) shouldn’t battle in importance with the street
    lines
- Street names are visible only on a second look
- Clean all unnecessary noise
Creative permissions
- Once you get it clean, simple, peaceful and easy
    to read you are allowed to infuse some style
- Charisma and personality is not the most
    important, it comes mostly from the rest of the
    layout and illustration
- Remember: the map is not an illustration, it is
    a tool. If you make it too cool and loud, people
    simply don’t trust it


# Drawing the street map

# Google maps (I)

1. Find your city on Google Maps. Choose “map”
    view (not satellite view, because all the info will
    confuse you).
2. Zoom in quite close, so that you can see the name
    of every street.
3. Now you have to make screenshots of the whole
    map area you need, with small overlaps between
    the images. Try to follow a “mental grid” (left
    to right, top to bottom), and be consistent;
    otherwise you will end up with a huge mess of
    images.
4. Save all the screenshots in one folder (this will
    make life easier for Adobe Illustator).


# Drawing the street map

# Google maps (II)

1. On Adobe Illustrator create a large canvas.
2. Import all the screenshots and give them some
    transparency (like 70% or so).
3. Compose the puzzle of screenshots in order to
    recreate the original Google Map.
4. You can also do this on Photoshop. This way you
    can better clean all the unecessary things on
    the screenshots (broweser windows, etc). Then
    import just one image into Illustrator.
5. Name this layer “Google Maps – vector”. Lock it.


# Drawing the street map

# Drawing the streets (I)

1. Create a new layer above “Google Maps –
    vector”, and name it “Streets”.
2. Lets be consistent and use not more than 4 stroke
    widths for the streets: 6pt, 8pt, 10pt and 12pt are
    a good reference, but it all depends on your exact
    map size.w
3. Give a different color to each stroke weight.
    This will make it much easier when you need to
    select them, and for you to know what you’re
    working with.
4. Create an “object style” for each street width.
    This way you won’t need to manually select it all
    the time.


# Drawing the street map

# Drawing the streets (II)

1. Start drawing the streets using the pen tool.
    Simply follow the Google Maps drawing. Try to
    simply complex curves, meaning trying to make
    a single stroke with as few points as possible.
    Also, draw one line per street (even if the street is
    longer; this will make it easier when you need to
    name the streets).
2. You can give the strokes some transparency. It
    probably will make your life easier in some cases.


# Drawing the street map

# Make a test print!

1. It might sound really obvious, but sometimes
    we forget that you can’t really see on the screen
    what’s going on. Make test prints along the way,
    before moving onto the next step of the work.
    You don’t want bad surprises at the end. Black
    and white simple prints are more than enough, no
    need to go fancy.


# Drawing the street map

# Open spaces, parks and water

1. Once you’ve got all the streets, lock layer
    “Streets”. Now you will draw squares. Create a
    new layer bellow the previous one, and name it
    “Open Spaces”. Draw squares and open spaces
    using a closed shape with no stroke. Give them a
    colour as well.
2. Do the same for parks (in a new layer named
    “Green Spaces”).
3. And again the same for river and lakes (name the
    layer “Water”). Use different colours.


# Drawing the street map

# Final steps

1. Now lets create the final street map: lock all
    layers except for: “Streets” and “Open Spaces”.
2. Select everything on both layers (ctrl+A) and copy
    (ctrl+C).
3. Create a new layer and name it: “MAP (merged)”
    (this will be the most important layer).
4. Into this new layer, paste-in-place (shift+ctrl+V)
    everything you had copied.
5. With everything still selected go to “object” and
    click “expand”.
6. With everything still selected go to “pathfinder”
    and click “merge”.
7. Now you have a shape with all the streets and
    open spaces. Make it white, and give it a black
    0,3pt stroke. It’s done!
8. If you need to correct a street or a square, you
    can go back to the original layers, change them,
    and then just you have to repeat steps 4 to 7 and
    you’re good to go!


# Naming the streets

1. Duplicate the layer “Streets”. Change the name
    to “Street Names” and put it above all the other
    layers. Give it some transparency.
2. Make the “MAP (merged)” layer invisible.
3. Now you can see the street names on Google
    Maps through layers “Streets” and “Street
    Names”. Make sure all layers are locked but for
    “Street Names”.
4. Click long enough on the “Type” icon and a menu
    will appear; choose “Type on a Path”
5. Click on a street you want to name and type the
    name. It’s easy, just copy it from the google map
    bellow.


# Formating street names

1. Give the street name the size and font you like.
    It’s best and easiest if you create a character style
    for each street width. There’s no formula for font
    sizes. It all depends on the font you pick. Don’t
    forget to make test prints!
2. The street names won’t appear centred inside the
    line. Fix this once you’ve named all the streets.
    Select all the names and then go to “Type – Type
    on a Path – Type on a Path Option”. On “Align to
    Path” choose: “Centre”. And there you go.
3. Name the squares and parks as well. For those,
    just create a regular text frame.
4. To easily see which streets you’ve named
    and which you haven’t, just turn on the “MAP
    (merged)” layer and the streets you haven’t
    named yet will appear coloured.


# Final layers and optionals

1. In case you’ve got exceptions, like, say, a tunnel,
    or a street that is actually stairs, add a layer
    and call it “Exceptions”. This way, if you need to
    change things on the “MAP” layer, you won’t have
    problems.
2. Add more layers for “Transportation”, “Walks”
    and other things you might need.
3. Add a layer with a background colour. It should be
    a neutral light colour.
4. Switch off all the layers with info that you don’t
    need (!!!DON’T DELETE THE LAYERS!!!) and
    voilà! you’ve got a fully usable map, that you can
    update anytime you need.
5. You then just have to place this map in InDesign,
    and carry on with your layout from there. For
    this, in InDesing, go to “Place” and choose the
    Illustrator file. Don’t attempt at copying layers
    from this Illustrator map into InDesign because
    you will get a huge mess.
6. Save the map for future use. It’s beautiful, easy to
    edit, and yours for ever.
